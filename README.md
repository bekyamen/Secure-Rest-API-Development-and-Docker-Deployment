# Customer Management System

A Node.js + Express + Prisma 7 + PostgreSQL application to manage customer lists.  
Each user can only access and manage their own customers. The app uses JWT authentication and is fully Dockerized.

---

## Features

- User registration and login with JWT authentication  
- User-isolated customer CRUD (Create, Read, Update, Delete)  
- PostgreSQL database with Prisma 7 ORM  
- Fully Dockerized: app and database run via Docker Compose  
- ES Modules compatible

---

## Tech Stack

- Node.js v24  
- Express.js v5  
- Prisma 7  
- PostgreSQL 16  
- Docker + Docker Compose  
- bcrypt for password hashing  
- jsonwebtoken for authentication
